# Student Admission System 🎓

A web-based student admission system built using HTML, CSS, PHP, and MySQL. It collects student information through a form and stores the data in a MySQL database.

## Features
- Admission form to collect student details
- Stores data including:
  - Name
  - Email
  - Phone Number
  - Course
  - Gender
  - Location
- Data saved securely in a MySQL database

## Technologies Used
- HTML
- CSS
- PHP
- MySQL

## How to Run
1. Place the project folder inside htdocs (if using XAMPP)
2. Start Apache and MySQL from XAMPP Control Panel
3. Import the database file into phpMyAdmin
4. Open the project in browser via localhost/student-admission-system
